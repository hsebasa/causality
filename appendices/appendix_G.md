**G. The Internal Causal Language.** *How to extend the object language so that causation is itself an event, and arbitrary nested causal expressions denote elements of $L$ — using only the one law $f$ and the lattice, with no second law, no quantale, and no complements. The relation "$A$ causes $B$" (1.11′, 1.16) returns a judgment, not an event, so it cannot be re-fed into another causal claim. The fix is to internalize causation as a binary connective $L \times L \to L$ built from $f$.*

**G.1 Definition (strict cone; the causal connective).** Write $f^{+}(A) := \bigvee_{n \geq 1} f^{n}(A)$ for the **strict future cone** of $A$ (Def. 1.16's latency-$\geq 1$ aggregate). Equivalently $f^{+} = f \circ f^{\ast} = f^{\ast} \circ f$ (from $f^{\ast} = \bigvee_{n \geq 0} f^{n}$, Thm. 1.14). It is **join-preserving** (a join of the join-preserving $f^{n}$) and monotone, but *not* extensive: $A \leq f^{+}(A)$ may fail (in translation, $f^{+}\{0\} = \{1,2,\dots\} \not\ni 0$) — that is the point of excluding $n = 0$. Define the **causal connective**
$$A \triangleright B \;:=\; f^{+}(A) \wedge B \qquad (\text{``the occurrence of } B \text{ that } A \text{ brings about''}),$$
so $\triangleright : L \times L \to L$. It decomposes by latency: with $A \triangleright_{n} B := f^{n}(A) \wedge B$ one has $A \triangleright B = \bigvee_{n \geq 1} (A \triangleright_{n} B)$ — assembled entirely from powers of $f$. The **reflexive variant** $A \triangleright^{\!\ast} B := f^{\ast}(A) \wedge B$ uses $f^{\ast}$ instead (includes the present); it gives $A \triangleright^{\!\ast} A = A$, the preorder reading $\sqsubseteq$ of 1.11′. We take the strict $\triangleright$ as primary, matching "produces" (1.16).

**G.2 Proposition (boundary reductions).** For all $A, B \in L$: *(a)* $A \triangleright B \leq B$ — the caused part of $B$ is part of $B$. *(b)* $A \triangleright B = \bot \iff A$ does **not** produce $B$ (1.16). *(c)* $A \triangleright B = B \iff B \leq f^{+}(A) \iff A$ strictly produces *all* of $B$. *(d)* $\bot \triangleright B = \bot = A \triangleright \bot$ (nothing from nothing, 1.6a). *Proof.* (a) is $f^{+}(A) \wedge B \leq B$. For (b), in a frame $f^{+}(A) \wedge B = \big(\bigvee_{n \geq 1} f^{n}(A)\big) \wedge B = \bigvee_{n \geq 1}\big(f^{n}(A) \wedge B\big)$, which is $\bot$ iff every $f^{n}(A) \wedge B = \bot$, i.e. $A$ produces no part of $B$. (c) is $f^{+}(A) \wedge B = B \iff B \leq f^{+}(A)$. (d) uses $f^{+}(\bot) = \bigvee_{n\geq 1} f^{n}(\bot) = \bot$ (1.6a) and $\,\cdot \wedge \bot = \bot$. $\square$ So the two Boolean endpoints of the relation — "produces" and "produces all" — are recovered as $\neq \bot$ and $= B$; the event $A \triangleright B$ is the **graded** answer interpolating them (the relations are recovered formally in G.10–G.11).

**G.3 Definition + Theorem (the language and its denotation).** Fix a set $\mathrm{At} \subseteq L$ of atomic event-names. **Terms** are generated by
$$t \;::=\; a \;\mid\; t \vee t \;\mid\; t \wedge t \;\mid\; t \triangleright t \qquad (a \in \mathrm{At}),$$
reading $\vee$ as "or", $\wedge$ as "and", $\triangleright$ as "causes". The **denotation** $\llbracket\,\cdot\,\rrbracket : \mathrm{Term} \to L$ is defined by structural recursion:
$$\llbracket a \rrbracket = a,\quad \llbracket s \vee t \rrbracket = \llbracket s \rrbracket \vee \llbracket t \rrbracket,\quad \llbracket s \wedge t \rrbracket = \llbracket s \rrbracket \wedge \llbracket t \rrbracket,\quad \llbracket s \triangleright t \rrbracket = f^{+}\!\big(\llbracket s \rrbracket\big) \wedge \llbracket t \rrbracket.$$
**Theorem (closure).** *$\llbracket t \rrbracket \in L$ for every term $t$; hence $L$ is closed under causation and every nested causal expression denotes an event.* *Proof.* Induction on $t$: the base case is $a \in L$, and each clause applies an operation $L \times L \to L$ ($\vee$, $\wedge$, or $\triangleright$). $\square$ **Queries.** A term $t = s \triangleright u$ is *fully realized* iff $\llbracket t \rrbracket = \llbracket u \rrbracket$ (G.2c) and *possible* iff $\llbracket t \rrbracket \neq \bot$ (G.2b); the event $\llbracket t \rrbracket$ carries strictly more information than either Boolean collapse.

**G.4 Proposition (algebra of $\triangleright$).** In a frame $L$ (e.g. the canonical $\mathcal{P}(X)$, Def. 1.4): *(a)* $\triangleright$ is monotone in both arguments. *(b)* **Cause-slot join-preservation:** $\big(\bigvee_{i} A_{i}\big) \triangleright B = \bigvee_{i} (A_{i} \triangleright B)$. *(c)* **Cause-slot sub-meet only:** $\big(\bigwedge_{i} A_{i}\big) \triangleright B \leq \bigwedge_{i} (A_{i} \triangleright B)$, generally strict. *(d)* **Effect-slot linearity:** $A \triangleright \big(\bigvee_{i} B_{i}\big) = \bigvee_{i} (A \triangleright B_{i})$ and $A \triangleright \big(\bigwedge_{i} B_{i}\big) = \bigwedge_{i} (A \triangleright B_{i})$. *(e)* **Non-associativity:** $(A \triangleright B) \triangleright C = f^{+}(f^{+}(A) \wedge B) \wedge C$, whereas $A \triangleright (B \triangleright C) = f^{+}(A) \wedge f^{+}(B) \wedge C$; these differ in general. *Proof.* (a) from monotonicity of $f^{+}$ and $\wedge$. (b) $f^{+}(\bigvee A_{i}) \wedge B = (\bigvee f^{+}A_{i}) \wedge B = \bigvee(f^{+}A_{i} \wedge B)$, using join-preservation of $f^{+}$ then frame distributivity. (c) from $f^{+}(\bigwedge A_{i}) \leq \bigwedge f^{+}A_{i}$ (1.6c); strictness in G.6. (d) the effect slot is "$\wedge$ with the constant $f^{+}A$", which preserves joins (frame) and meets (in any complete lattice). (e) compute; witness in G.6. $\square$ **Reading.** The cause-slot split (b valid, c not) is the existential-valid / universal-invalid asymmetry of the quantifier interlude (*Causation and the Quantifiers*), now internal to a single connective; non-associativity (e) is why parenthesization in a nested expression is semantically load-bearing.

**G.5 Definition (the cause-side dual).** Because $f^{+}$ preserves arbitrary joins, it has a right adjoint $(f^{+})^{\sharp}(B) := \bigvee\{A : f^{+}(A) \leq B\}$ (the construction of $f^{\sharp}$, Ch. 2, applied to $f^{+}$); it preserves arbitrary meets. Define the **abductive connective**
$$A \triangleleft B \;:=\; A \wedge (f^{+})^{\sharp}(B) \qquad (\text{``the part of } A \text{ whose entire future lies in } B\text{''}).$$
By the adjunction $f^{+}(A) \leq B \iff A \leq (f^{+})^{\sharp}(B)$, so $A \triangleright B = B \iff A \triangleleft B = A$: the two connectives certify the same full-causation fact from the effect side and the cause side. Where $\triangleright$ is existential in its active slot, $\triangleleft$ is universal (meet-preserving) — production versus explanation (Ch. 2), internalized. The grammar of G.3 may be extended with $t \triangleleft t$, $\llbracket s \triangleleft t \rrbracket = \llbracket s \rrbracket \wedge (f^{+})^{\sharp}\llbracket t \rrbracket$, preserving closure.

**G.6 Worked evaluations.** *(i) The general expression.* Outside-in, $\llbracket (A \triangleright X) \triangleright (B \triangleright (C \triangleright (D \vee (H \triangleright F)))) \rrbracket = f^{+}\!\big(f^{+}(A) \wedge X\big) \wedge f^{+}(B) \wedge f^{+}(C) \wedge \big(D \vee (f^{+}(H) \wedge F)\big)$, a single element of $L$. *(ii) Strictness of G.4(c) (a merging law).* $X = \{p,q,r,z\}$, $f(p)=p$, $f(q)=f(r)=z$, $f(z)=z$; then $f^{+}\{p,q\} = \{p,z\}$, $f^{+}\{p,r\} = \{p,z\}$, and with $B = \{z\}$: $(\{p,q\} \wedge \{p,r\}) \triangleright B = f^{+}\{p\} \wedge \{z\} = \{p\} \wedge \{z\} = \bot$, while $(\{p,q\} \triangleright B) \wedge (\{p,r\} \triangleright B) = \{z\} \wedge \{z\} = \{z\}$ — strict, the same merging that breaks universal quantification. *(iii) Non-associativity (a path law).* $X = \{0,1,2,3,4\}$, $f(i) = \min(i{+}1,4)$; with $A = \{0\}$, $B = \{0,2\}$, $C = \{1\}$: $(A \triangleright B) \triangleright C = f^{+}(\{1,2,3,4\} \wedge \{0,2\}) \wedge \{1\} = f^{+}\{2\} \wedge \{1\} = \{3,4\} \wedge \{1\} = \bot$, whereas $A \triangleright (B \triangleright C) = f^{+}\{0\} \wedge \big(f^{+}\{0,2\} \wedge \{1\}\big) = \{1,2,3,4\} \wedge \{1\} = \{1\}$. Distinct, so the nesting is real.

**G.7 Remarks (invariance; what the fragment is).** *(a) Transport.* A morphism $h$ of $\mathbf{Caus}$ (*The Category of Causal Structures*) preserves $\vee$ and intertwines $f$, hence $f^{+}$, but need not preserve $\wedge$; so it need not preserve $\triangleright$. It does when $h$ preserves meets — i.e. for non-merging (injective) simulations, in particular **$\mathbf{Caus}$-isomorphisms** — the same injectivity condition that rescues $\forall$ in the quantifier interlude. Thus $\llbracket\,\cdot\,\rrbracket$ is invariant along causal isomorphisms. *(b) Refinement.* $\triangleright$ is the event-valued refinement of two existing notions: $A \triangleright B \neq \bot$ is the production relation (1.16), and $A \triangleright^{\!\ast} A = A$ recovers the causal preorder $\sqsubseteq$ (1.11′). *(c) Honest accounting.* The language is the **positive, single-law fragment**: it never leaves $L$, introduces no second law (contrast the regime-change reading, which folds the law into the state, 1.18), and uses no quantale residuation or complement (contrast the internal-hom reading; "absences are not events", 1.4, 1.22). Everything is one meet away from the one axiom.

**G.8 Definition (latency-graded operators).** For $n \in \mathbb{N}$ and $A, B \in L$ set $A \triangleright_{n} B := f^{n}(A) \wedge B$, with the cause-only abbreviation $\mathsf{F}_{n}(A) := A \triangleright_{n} \top = f^{n}(A)$. Then $\triangleright_{0} = {\wedge}$ (since $f^{0} = \mathrm{id}$), the strict connective of G.1 is $A \triangleright B = \bigvee_{n \geq 1} A \triangleright_{n} B$, and the reflexive one is $A \triangleright^{\!\ast} B = \bigvee_{n \geq 0} A \triangleright_{n} B$. The index $n$ is the latency of Def. 1.16; the family $(\mathsf{F}_{n})_{n}$ is the $\mathbb{N}$-action of App. A ($\mathbb{N}$ free on one generator — the law).

**G.9 Proposition (algebra of the graded operators).** *(a) Diagonal:* $A \triangleright_{0} B = A \wedge B$ — pure overlap, no dynamics (why 1.16 starts the count at $n = 1$). *(b) Slotwise linearity:* each $\triangleright_{n}$ preserves joins in the cause slot and is fully linear (preserves $\bigvee$ and $\bigwedge$) in the effect slot, the cause-slot meet being merely sub-additive, $(\bigwedge_{i} A_{i}) \triangleright_{n} B \leq \bigwedge_{i}(A_{i} \triangleright_{n} B)$ — the asymmetry of G.4, uniformly in $n$. *(c) Latency additivity:* $\mathsf{F}_{m} \circ \mathsf{F}_{n} = \mathsf{F}_{m+n}$, so $n \mapsto \mathsf{F}_{n}$ is a monoid homomorphism $(\mathbb{N}, +, 0) \to (\mathrm{End}_{\bigvee} L, \circ, \mathrm{id})$; equivalently the **chained-production identity** $(A \triangleright_{n} \top) \triangleright_{m} B = A \triangleright_{m+n} B$ holds for all $m, n$. *Proof.* (a) $f^{0} = \mathrm{id}$. (b) $f^{n}$ is join-preserving (a power of $f$); the effect slot is $\wedge$ with the constant $\mathsf{F}_{n}(A)$, and meet with a constant preserves all meets in any complete lattice and all joins in a frame; the cause-slot sub-meet is 1.6c applied to $f^{n}$. (c) $\mathsf{F}_{m}(\mathsf{F}_{n}(A)) = f^{m}(f^{n}(A)) = f^{m+n}(A)$, and $(A \triangleright_{n} \top) \triangleright_{m} B = f^{m}(f^{n}(A) \wedge \top) \wedge B = f^{m+n}(A) \wedge B = A \triangleright_{m+n} B$. $\square$

**G.10 Theorem (recovery from the graded family).** *From $(\triangleright_{n})_{n \in \mathbb{N}}$ and the lattice operations alone:* *(a) Powers and the law:* $f^{n}(A) = A \triangleright_{n} \top$; in particular $f = (\,\cdot\,) \triangleright_{1} \top$ and $\mathrm{id} = (\,\cdot\,) \triangleright_{0} \top$. *(b) Cones:* $f^{+}(A) = \bigvee_{n \geq 1}(A \triangleright_{n} \top)$ and $f^{\ast}(A) = \bigvee_{n \geq 0}(A \triangleright_{n} \top)$. *(c) Production and latency (1.16):* $A$ produces $B \iff \exists\, n \geq 1:\ A \triangleright_{n} B \neq \bot$, with $\tau(A, B) = \min\{\, n \geq 1 : A \triangleright_{n} B \neq \bot \,\}$. *(d) Causal order (1.11′):* $A \sqsubseteq B \iff B \leq \bigvee_{n \geq 0}(A \triangleright_{n} \top)$. *(e) Full production:* $A \triangleright B = B \iff B \leq \bigvee_{n \geq 1}(A \triangleright_{n} \top)$. *Proof.* (a) $A \triangleright_{n} \top = f^{n}(A) \wedge \top = f^{n}(A)$. (b) the definitions of $f^{+}, f^{\ast}$ together with Thm. 1.14. (c) $A \triangleright_{n} B = f^{n}(A) \wedge B$, so the statement is Def. 1.16 verbatim. (d) $\bigvee_{n \geq 0} f^{n}(A) = f^{\ast}(A)$ (Thm. 1.14) and $B \leq f^{\ast}(A)$ is Def. 1.11′. (e) $f^{+}(A) \wedge B = B \iff B \leq f^{+}(A)$. $\square$ Since (a) returns $f$ itself, the graded family determines $(L, f)$ completely — and hence, through Ch. 1, every causal notion of the theory.

**G.11 Theorem (the invariant tower; faithfulness).** *Compare causal structures on a fixed frame $L$ by the data they induce.* *(a) The graded family is faithful:* if $A \triangleright^{f}_{n} B = A \triangleright^{g}_{n} B$ for all $n, A, B$, then $f = g$. *(b) The aggregate is a proper quotient:* $\triangleright^{f} = \triangleright^{g} \iff f^{+} = g^{+}$, which does **not** force $f = g$. From $\triangleright$ alone one still recovers $\mathsf{F}(A) := A \triangleright \top = f^{+}(A)$, $\mathsf{R}(A) := A \vee (A \triangleright \top) = f^{\ast}(A)$, production ($A \triangleright B \neq \bot$, 1.16) and the causal order ($B \leq \mathsf{R}(A)$, 1.11′) — but not the one-step law. *Proof.* (a) By G.10(a), $f(A) = A \triangleright^{f}_{1} \top = A \triangleright^{g}_{1} \top = g(A)$ for every $A$. (b) $A \triangleright^{f} B = f^{+}(A) \wedge B$, so equality for all $A, B$ is, on taking $B = \top$, equivalent to $f^{+} = g^{+}$; the recoveries are G.10(b,c,d) read off the aggregate. For non-faithfulness, on $X = \{a, b, c\}$ with $c$ a sink take $f(a) = b,\ f(b) = c$ and $g(a) = b \vee c,\ g(b) = c$ (both fixing $c$): both are join-preserving and distinct ($f(a) \neq g(a)$), yet each sends $a \mapsto \{b,c\}$, $b \mapsto \{c\}$, $c \mapsto \{c\}$ under $f^{+}$ resp. $g^{+}$, so $f^{+} = g^{+}$ and $\triangleright^{f} = \triangleright^{g}$. $\square$ **The tower.** $f$ (finest, from $\triangleright_{1}$) $\rightsquigarrow f^{+}$ (from $\triangleright$) $\rightsquigarrow$ production and $\sqsubseteq$: the powers collapse to their join, the join collapses to a $\neq \bot$ test, and each forgetting is strict.
